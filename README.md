# String Slicing Demo in Python

This repository contains a beginner-friendly Python program that demonstrates string slicing using the name `dhananjay`.

## What this project shows
- Full string printing
- Positive slicing
- Negative slicing
- Extracting first and last characters
- Understanding Python string indexes

## Example output
```python
name = "dhananjay"
print("Complete name:", name)
print("From 2nd letter:", name[1:])
print("From 3rd letter:", name[2:])
print("From 4th letter:", name[3:])
print("Last letter:", name[-1:])
print("Last two letters:", name[-2:])
print("Last three letters:", name[-3:])
print("Last four letters:", name[-4:])
print("First 5 letters:", name[0:5])
print("First 6 letters:", name[0:6])
print("All except last 5 letters:", name[:-5])
print("All except last 6 letters:", name[:-6])
print("All except last 7 letters:", name[:-7])
print("All except last 8 letters:", name[:-8])
```

## How to run
1. Save the code in a file named `slicing_demo.py`.
2. Run it using Python:
   ```bash
   python slicing_demo.py
   ```
